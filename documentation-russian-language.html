Подсветка кода (Prism)

Для подсветки кода в этой документации используется библиотека Prism. Ссылка: https://prismjs.com/. Инструкции по использованию: https://prismjs.com/index.html#basic-usage.

  • Тема (или цветовая схема Prism): Default (тема по умолчанию, светлая тема);
  • Поддерживаемые языки: Markup + HTML + XML + SVG + MathML + SSML + Atom + RSS, CSS, C-like, JavaScript, .ignore + .gitignore + .hgignore + .npmignore, JSON + Web App Manifest, Markdown, Markup templating, PHP, SQL (Это список выбранных языков на странице скачивания PrismJS);
  • Подключённые плагины: Line Highlight (Подсветка строки), Line Numbers (Нумерация строк), Toolbar (Панель инструментов; Плагин нужен для работы кнопки "Копировать код"), Copy to Clipboard Button (Кнопка "Копировать код") (Это список выбранных плагинов на странице скачивания PrismJS).

Если вам нужна поддержка большего количества языков, или вы хотите использовать другую тему для подсветки кода (например, тёмную тему), то вы всегда можете выбрать нужные вам языки и тему на странице скачивания Prism здесь: https://prismjs.com/download.html. После скачивания вашей версии библиотеки Prism (это файлы prism.css и prism.js), вам просто нужно заменить файлы Prism документации вашими новыми файлами (расположение файлов для замены в документации: /assets/css/prism.css и /assets/js/prism.js).

			social_tools: false,
